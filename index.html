Pollarity is a web app that visualizes trends in state and national public opinion polls and attempts to predict future poll results. It uses data from the Huffington Post Pollster API, which is processed and served by a Rails back end to the Backbone.js-powered front end. It uses D3.js to bring that data to life in animated line graphs, bar graphs, and an interactive map of the US.

Technologies: Ruby on Rails, Backbone.js, D3.js, Datamaps, jQuery UI, HTML/CSS
Troll Hunter is a collection of the most provocative reviews of Chicago restaurants found on Yelp. We scraped Yelp's website and scored reviews using a hilarious-keyword-matching algorithm.

Technologies: Ruby, Sintara, Nokogiri, HTML/CSS
